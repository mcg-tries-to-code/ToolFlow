# Examples

Reference workflows and generated artifacts for ToolFlow.

Current example workflows:
- `workflows/safe-profile-mvp.json` - basic read/research/session flow
- `workflows/patch-after-approval.yaml` - elevated patch flow with explicit approval pause
- `workflows/repo-diagnostics.yaml` - repository diagnostics workflow template
- `workflows/research-digest.yaml` - research-and-summary workflow template
- `workflows/session-fanout.yaml` - multi-session investigation workflow template
- `workflows/native-taskflow-architecture.json` - generic controller and TaskFlow orchestration example

Removed from the public repo surface:
- workspace-specific and project-specific example workflows that referenced local paths, private project structures, or operator-specific environments
